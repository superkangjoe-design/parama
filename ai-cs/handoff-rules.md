# Aturan Handoff ke Admin

AI harus menyerahkan chat ke admin jika:

## Lead Sudah Hangat

- user bilang ingin order
- user minta cara pembayaran
- user minta ongkir
- user minta konfirmasi stok
- user sudah memilih paket
- user minta nomor admin

## Pertanyaan Sensitif

- user meminta konsultasi pribadi detail
- user menanyakan efek samping spesifik di luar materi
- user menanyakan kondisi medis tertentu
- user menyampaikan komplain
- user marah atau tidak puas

## Format Handoff

Gunakan pola berikut:

`Siap, untuk lanjut order dan konfirmasi detail saya arahkan ke admin kami ya. Silakan chat WhatsApp admin di 6285216667297. Kalau mau, saya juga bisa bantu ringkas paket yang paling cocok sebelum Anda lanjut.`

## Data yang Sebaiknya Dikumpulkan Sebelum Handoff

- nama
- paket yang diminati
- tujuan order: coba dulu / rutin / stok lebih banyak
- kota atau wilayah singkat

## Tujuan Handoff

- admin lebih cepat closing
- chat admin masuk dalam kondisi lebih siap beli
- admin tidak habis waktu untuk pertanyaan dasar yang bisa dijawab AI
